Crest

Muvendar Seal (மூவேந்தர்)

Unites the three ancient kingdoms: Chera (Bow), Chola (Tiger), and Pandya (Twin Fish). Adds unmatched prestige.

Auspicious

Sikku Kolam (கோலம்)

Mathematical symmetrical geometry drawn at thresholds. Represents hospitality, learning, and order.

Enlighten

Kuthuvilakku (விளக்கு)

Traditional pedestal lamp ignited to dispel ignorance. Perfectly fits landing pages for educational organizations.

Botanical

Senkaanthal (செங்காந்தள்)

The Flame Lily. Highly praised in Sangam poetry representing passionate learning and ecological resilience.

Codebase Implementation Guide

Follow these simple steps to safely implement these patterns inside the React codebase:

1

Import Fonts

Add the Google Fonts link inside the <head> element in index.html (or import them via CSS):

                            @import url('https://fonts.googleapis.com/...');
2

Tailwind Setup

Map the chosen fonts under fontFamily inside tailwind.config.js so they can be accessed dynamically:

                            fontFamily: {
                            tamilHeading: ['Kavivanar', 'cursive'],
                            // or 'Baloo Thambi 2', 'Arima', 'Pavanam'
                            }
3

Apply Globally

Add standard styles inside src/index.css to default your headers to the cultural typeface:

h1,
                            h2, h3, h4 {
                            @apply font-tamilHeading;
                            }
